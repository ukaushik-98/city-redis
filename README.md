# City Redis
Personal implementation of Redis DB in Rust.

## Purpose
Learn how Redis works. Covers the following Redis features:
- [ ] Async Replication
- [ ] Buffer Parsing
- [ ] Replication
- [ ] Commands
  - [ ] PING
  - [ ] GET
  - [ ] SET
  - [ ] PUBLISH
  - [ ] SUBSCRIBE
- [ ] CMD Interface
- [ ] OpenTelemetry

## Dependencies
- Tokio
